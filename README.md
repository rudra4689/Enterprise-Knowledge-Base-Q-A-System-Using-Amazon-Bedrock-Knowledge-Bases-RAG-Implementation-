# 🚀 Enterprise Knowledge Base Q&A System (RAG) using Amazon Bedrock

An intelligent Question-Answering system built using **Retrieval-Augmented Generation (RAG)** with **Amazon Bedrock Knowledge Bases**.  
This system allows users to ask natural language questions on private enterprise data and receive accurate, citation-backed responses.

---

## 📌 Project Overview

Traditional enterprise search systems rely on keyword matching and often fail to understand context.  
Large Language Models (LLMs) may generate incorrect answers (hallucinations) when working with private or recent data.

👉 This project solves both problems using **RAG architecture**, combining:
- 🔍 Semantic Search (Vector Retrieval)
- 🧠 LLM-based Answer Generation

---

## 💡 Features

- ✅ Ask questions in natural language  
- ✅ Accurate answers based on internal documents  
- ✅ Citation-backed responses (reduces hallucinations)  
- ✅ Semantic search using vector embeddings  
- ✅ Secure handling of private enterprise data  
- ✅ Interactive UI using Streamlit

---

## 🏗️ System Architecture (Flow)

1. User asks a question  
2. Query is converted into embeddings  
3. Relevant documents are retrieved from Knowledge Base  
4. LLM generates answer using retrieved context  
5. Response is returned with citations
